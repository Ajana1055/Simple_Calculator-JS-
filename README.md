🧮 Simple JavaScript Calculator
A lightweight, responsive calculator built entirely with Vanilla JavaScript, HTML5, and CSS3. This project demonstrates the power of functional programming and DOM event handling without the need for external libraries or frameworks.

🚀 Features
Basic Arithmetic: Supports addition, subtraction, multiplication, and division.

Decimal Precision: Handles floating-point calculations accurately.

Clear & Delete: Includes functionality to clear the entire display or delete the last entered digit.

Keyboard Support: Fully functional via mouse clicks or numeric keypad input.

Responsive Design: Optimized for both desktop and mobile screens.

🛠️ Tech Stack
HTML5: Semantic structure for the calculator interface.

CSS3: Custom styling with Flexbox/Grid for a modern look.

JavaScript (ES6+): Logic for calculations, event listeners, and UI updates.
